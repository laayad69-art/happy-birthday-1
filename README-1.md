## 🚀 طريقة الاستخدام

### 1. تشغيل الموقع
فتح ملف `alaa_birthday.html` مباشرة في أي متصفح — مفيش سيرفر محتاج.

### 2. كلمة السر
```
3/16
```

### 3. إضافة الفيديوهات
حط ملفات الفيديو في **نفس مجلد** ملف الـ HTML بالأسماء دي:
```
📁 المجلد
├── alaa_birthday.html
├── alaa1.mp4          ← فيديو ١
└── alaa2.mp4          ← فيديو ٢
```

---

## 📁 هيكل المشروع

```
alaa_birthday.html     ← ملف واحد كامل (HTML + CSS + JS + صور + موسيقى)
alaa1.mp4              ← فيديو ١ (اختياري)
alaa2.mp4              ← فيديو ٢ (اختياري)
README.md              ← أنت هنا
```

> **ملاحظة:** الصور والموسيقى مدمجين داخل الـ HTML كـ Base64 — الملف شغال standalone بدون أي ملفات خارجية عدا الفيديوهات.
